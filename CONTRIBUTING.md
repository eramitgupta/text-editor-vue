# Contributing

Thanks for your interest in contributing!

## Bug Reports

Open an [issue](https://github.com/eramitgupta/text-editor-vue/issues) with steps to reproduce, expected vs actual behavior, and your browser/Vue/package versions.

## Pull Requests

1. Fork & branch from `main`.
2. Make changes in `src/`.
3. Run checks:
   ```bash
   npm run lint && npm run typecheck && npm run build
   ```
4. Open a PR with a clear description.

## Development

```bash
npm install
npm run dev       # watch mode
npm run format    # format code
```
