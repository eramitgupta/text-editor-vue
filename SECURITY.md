# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | ✅         |
| < Latest | ❌        |

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Email the project maintainers with a description of the vulnerability, steps to reproduce, and any potential impact. You will receive a response within 48 hours.
